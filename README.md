# CSV ファイル注文処理ソフト
ツール名：Nations　CSV一括処理
ツールテストPC：Windows
ツールダウンロード公開：１１月１３日まで

# 特徴

  - 表示項目変更、再表示可能
  - ソフトをPCにインストールしない（コピーのみ）
  - ソフト軽い
  - マニュアル付き
  - 操作簡単
  - 今後画像一括編集、商品名変更などの利用も可能
